# plscheme
使用C++完成的scheme interpreter
### 使用方法
編譯 ```make ```

只有讀入功能：```./pl1.out ```

執行scheme interpreter:```./pl ```

test file : ```make test```
### 目前完成的功能
- function call 
- 數學四則運算
- string相關功能
- cell的type查詢
- 自定義變數
